# Algorithm-Visualizer

A Multi Algorithm Visualizer application which can be used to analyse various Algorithms:

## 1. Path Finding Algorithms:
   a. BFS
   b. DFS
   c. Dijkstra
   d. Heuristic

## 2. Sorting Algorithms:
   a. Bubble sort
   b. Merge sort
   c. Selection sort
   d. Insertion sort
   e. Quick sort
   
## 3. BST visualizer for Insertion and Deletion.
